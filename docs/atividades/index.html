Conferências

As 3 conferências serão feitas por convidados indicados pela comissão organizadora.

Apresentações orais

As apresentações orais serão espaços para desenvolvedores de pacotes apresentarem seus pacotes, criando espaço para o recebimento de contribuições, geração de parcerias e compromissos futuros, como o desenvolvimento coletivo de projetos sobre R. Cada sessão terá duração de 1 hora com 3 apresentações de 20 minutos cada, sendo os 5 minutos finais de cada uma dedicados à participação da audiência. Os conferencistas das apresentações orais serão selecionados mediante inscrição de propostas.

Tutoriais

Os tutoriais terão duração de 1 hora cada e serão realizados em dois segmentos simultâneos. Os tutoriais ímpares são voltados para os usuários buscando conhecimento em visualização e manipulação de dados, bem como ferramentas para editoração de código e pesquisa reproduzível. Já os tutoriais pares são voltados para os inclinados a desenvolver pacotes, e irá incluir ferramentas para o desenvolvimento de pacotes, recursos para paralelização de processamento e aplicações em Shiny. Os ministrantes dos tutoriais serão selecionados mediante inscrição de propostas.

Observação: Os temas e a quantidade de tutorias poderão sofrer alterações conforme as propostas que forem submetidas.
